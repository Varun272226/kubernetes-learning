# Kubernetes Learning 

A hands-on Kubernetes learning repository with practical examples, YAML manifests, interview notes, and production concepts.


---

##  About

This repository documents my hands-on Kubernetes learning journey from beginner to production-ready concepts.

Every topic includes theory, practical examples, YAML manifests, commands, interview questions, and notes prepared during my learning journey.

## 🎯 Learning Objectives

The primary objectives of this repository are:

- Learn Kubernetes fundamentals from scratch.
- Practice Kubernetes concepts using Minikube.
- Understand how Kubernetes works internally.
- Create and manage Kubernetes resources using YAML manifests.
- Prepare for Kubernetes and DevOps interviews.
- Build a professional GitHub portfolio.
